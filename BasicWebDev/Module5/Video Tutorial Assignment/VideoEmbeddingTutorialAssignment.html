Video Embedding Tutorial Assignment

Steps to embed a video on a website are:

1. Save video in the project directory file.

2. Open <video> tag in html document to establish video player. Assign height and width measurements for display size. Ensure that it does not exceed the height and width measurements of the original video file.

3. Apply controls attribute to <video> tag to display video controls such as volume control,time,etc. Set preload attribute to auto. This will ensure the video is fully or partially loaded by the browser before use.

4. Open a <source>tag nested within the <video> tag. Set file location in the src attribute. Set the media type of the file in the type attribute.

5. Add other filetypes of the same video in additional <source> tags. Adding more filetypes of the same video increases chances for finding a compatible video source.

6. Add a default text stating that the browser does not support the video. This text will be displayed in case none of the video files are supported by the browser.
